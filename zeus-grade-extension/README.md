# ZEUS Grade Watcher

ZEUS 성적 조회 화면만 감시하는 Chrome 확장입니다. 기존 Page Watcher처럼 DOM 영역을 선택하지 않고, ZEUS 성적 조회 요청인 `/ugd/ugdCptnMrksQ/select.do`의 `Dataset:dsMain` 응답을 파싱합니다.

## 사용 방법

1. Chrome 확장 프로그램 페이지에서 `zeus-grade-extension` 폴더를 압축해제된 확장으로 로드합니다.
2. ZEUS에 로그인하고 `개인성적조회`를 눌러 성적표가 보이는 화면을 엽니다.
3. 확장 아이콘을 누르고 `개인성적조회 화면 등록`을 클릭합니다.
4. `지금 확인`으로 초기 스냅샷을 저장합니다.
5. `핸드폰 등록 QR 표시`를 누르고 휴대폰 Chrome으로 QR을 열어 알림을 허용합니다.
6. `푸시 테스트`로 휴대폰 알림 수신을 확인합니다.
7. 필요하면 검사 주기를 고르고 `자동 감시 시작`을 누릅니다.

## 동작 확인

- 확장을 업데이트한 뒤에는 Chrome 확장 프로그램 페이지에서 확장을 새로고침하고, 열려 있던 ZEUS 탭도 한 번 새로고침하면 가장 확실합니다.
- 자동 감시가 켜지면 확장 아이콘에 `ON` 배지가 표시됩니다.
- 예약 검사가 성공하면 배지가 `OK`로 바뀝니다.
- 성적 변경이 감지되면 `NEW`, 검사 실패 시 `ERR` 배지가 표시됩니다.
- 성적 변경이 감지되면 Chrome 알림과 QR로 등록한 모바일 Web Push 알림이 함께 전송됩니다.
- 팝업을 다시 열면 `최근 결과`와 `마지막 확인` 시각을 볼 수 있습니다.
- 변경이 없으면 Chrome 알림은 뜨지 않고, 배지와 팝업 상태만 갱신됩니다.

## 어느 화면에서 등록하나요?

- 최초 등록은 개인성적조회 표와 학번이 보이는 화면에서 합니다.
- ZEUS 첫 화면의 `개인성적조회` 버튼은 성적 화면을 여는 버튼일 뿐, 확장 등록 대상은 아닙니다.
- 한 번 등록된 뒤에는 학번과 탭 정보가 저장되므로, 자동 감시는 ZEUS 탭이 열려 있고 로그인 세션이 유지되는 동안 동작합니다.
- 등록에 성공하면 팝업에 학번과 `개인성적조회 화면 등록 완료`가 표시됩니다.

## 동작 방식

- 등록된 ZEUS 탭 안의 content script가 같은 출처 요청으로 성적 조회 API를 호출합니다.
- 성적 조회 전 `/sys/main/refreshSessionTime.do`를 호출해 세션 시간을 연장합니다.
- 응답 SSV의 `dsMain` 데이터셋을 과목 단위 스냅샷으로 저장합니다.
- 이후 검사에서 학점, 등급, GPA 값이 바뀌면 Chrome 알림과 `watcher.kangbeen.my` Web Push 알림을 보냅니다.

## 제약

- ZEUS 로그인 탭은 열려 있어야 합니다.
- 탭을 앞으로 가져오거나 Nexacro 버튼을 클릭하지 않습니다.
- 전체 페이지 새로고침이나 ZEUS 상단 리프레쉬 버튼에 의존하지 않습니다.
- 서버 응답 형식이나 `pg_key`가 ZEUS에서 바뀌면 수정이 필요할 수 있습니다.
